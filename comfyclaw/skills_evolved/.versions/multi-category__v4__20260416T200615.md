# Multi-Category Object Generation

## When to Use
Trigger when the prompt contains multiple semantically distinct object categories:
- Animals + objects (pig and backpack, elephant and umbrella)
- Vehicles + nature elements
- Food + creatures
- Any combination where categories don't naturally co-occur

## Core Problem
Diffusion models struggle with multi-category scenes because:
1. Cross-contamination: Animal textures bleed onto objects
2. Dominance: One category suppresses the other
3. Attribute loss: Colors/materials get dropped when managing multiple objects
4. Spatial confusion: Objects merge or overlap incorrectly

## Strategy

### Step 1: Detect Attributes
Scan the prompt for color/material modifiers on ANY object:
- Colors: green, purple, brown, white, red, blue, etc.
- Materials: wooden, metal, glass, plastic, stone, etc.

If attributes are present, apply emphasis syntax immediately.

### Step 2: Attribute Emphasis
For each object with attributes, restructure as:
- `(color:1.3) object` → `(green:1.3) backpack`
- `(material:1.3) object` → `(wooden:1.3) flower`
- Combined: `(purple:1.3) (kangaroo:1.2)`

### Step 3: Regional Prompting Setup
Use regional-control skill to isolate categories:
- Region 1 (left/top): `(attribute:1.3) object1, white background`
- Region 2 (right/bottom): `object2, detailed, white background`
- Base: `object1 and object2, simple composition, studio lighting`

### Step 4: Negative Prompts
Add category-specific negatives:
- For objects: `animal fur, scales, feathers, organic texture`
- For animals: `plastic, manufactured, artificial, synthetic`

### Step 5:Sampler Tuning
- Increase CFG to 8.5-9.5 when attributes are present
- Use DPM++ 2M Karras sampler
- Steps: 35-40 (higher for complex attribute combinations)

## Example Transforms

**Input:** "a green backpack and a pig"
**Output prompt:**
```
Region 1: (green:1.4) (backpack:1.2), leather straps, zippers, white background
Region 2: pig, pink skin, detailed, white background  
Base: green backpack and pig, side by side, studio photo
Negative: animal fur on backpack, plastic pig, merged objects, blurry
```

**Input:** "a elephant and a purple kangaroo"
**Output prompt:**
```
Region 1: elephant, gray skin, tusks, detailed, white background
Region 2: (purple:1.4) (kangaroo:1.2), vibrant color, white background
Base: elephant and purple kangaroo, two animals, natural pose
Negative: purple elephant, gray kangaroo, color bleed, merged animals
```

## Integration Points
- Always combine with regional-control for spatial separation
- Call unusual-attributes if attributes are highly atypical (blue oranges, glass animals)
- Use counting-objects if count > 2
- Pair with photorealistic or creative based on user intent

## Success Criteria
- Both categories present in output
- Attributes correctly applied to target object only
- No texture/color contamination between categories
- Clear spatial separation maintained
