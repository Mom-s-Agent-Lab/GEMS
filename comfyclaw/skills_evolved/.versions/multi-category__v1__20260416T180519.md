# Multi-Category Object Generation

## When to Use
Trigger when the prompt contains two or more semantically distinct object categories in the same scene:
- Animal + inanimate object ("pig and backpack", "rabbits and donut")
- Vehicle + nature/animal ("cars and kangaroo", "car and flowers")
- Multiple animals of different species ("rabbits and sheep", "bears and donut")
- Food + creatures ("donut and bears")

Do NOT use for multiple instances of the same category (use counting-objects instead).

## Core Strategy

### 1. Category Isolation Syntax
Restructure the prompt to explicitly separate categories:
```
Original: "a green backpack and a pig"
Restructured: "[a green backpack:0.8], [a pig:0.8], two distinct objects, separate entities"
```

### 2. Anti-Contamination Tokens
Add negative prompts to prevent category blending:
```
negative: "hybrid, merged, blended, fused, combined creature, mutant, chimera, morphed"
```

### 3. Use Regional Prompting When Available
If regional-control is available, invoke it to assign each category to a distinct spatial zone:
- Left/right split for two categories
- Foreground/background for mixed scales

### 4. Explicit Count and Category Labels
Always specify:
- Total count: "two objects", "seven animals", "eight items"
- Category names: "mammal", "vehicle", "plant", "accessory"
- Separation phrase: "distinct", "separate", "individual"

Example:
```
Prompt: "four rabbits and a sheep, five animals total, distinct mammals, separated spacing"
Negative: "hybrid animal, merged creatures, chimera, blended fur"
```

### 5. CFG and Attention Tuning
- Increase CFG to 8-10 (stronger prompt adherence)
- Use attention emphasis on category-defining words: "(rabbit:1.2)", "(sheep:1.2)"

## Node-Level Implementation
1. Detect category split in prompt
2. Call regional-control if multiple objects detected
3. Restructure prompt with isolation syntax
4. Add category-specific negative prompts
5. Set CFG >= 8.0 in KSampler
6. If regional prompting unavailable, rely on spatial keywords ("on the left", "in the background")

## Example Transformations

**Input:** "a green backpack and a pig"
**Output:**
- Prompt: "[a green backpack:0.9] on the left, [a pink pig:0.9] on the right, two separate objects, distinct items, realistic photo"
- Negative: "hybrid, merged, pig-shaped backpack, backpack with animal features, chimera"
- CFG: 9.0

**Input:** "six cars and a kangaroo"
**Output:**
- Prompt: "six cars in the background, (kangaroo:1.2) in the foreground, seven total objects, vehicle and animal, distinct categories"
- Negative: "car-kangaroo hybrid, mechanical animal, vehicle with legs"
- Use regional-control: background region="six cars", foreground region="kangaroo"
