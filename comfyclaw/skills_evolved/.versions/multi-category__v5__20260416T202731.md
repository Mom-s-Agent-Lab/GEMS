# Multi-Category Object Generation

## When to Use
Trigger when the prompt contains objects from 2+ distinct semantic categories:
- Animals + inanimate objects (pig + backpack, raccoon + flower)
- Food + creatures (donut + bears, bagel + toy)
- Vehicles + nature (car + flowers)
- Any combination where categories have different material properties or typical contexts

## Core Problem
Diffusion models struggle when semantically distant categories share a scene because:
1. Training data rarely shows these combinations
2. Cross-attention leaks attributes between objects
3. One category often dominates and suppresses the other
4. Count enforcement fails when objects are conceptually unrelated

## Required Steps

### 1. Parse and Isolate
Extract each object with its attributes:
- "a green backpack and a pig" → Object1: "green backpack" (1x), Object2: "pig" (1x)
- "five bears and a donut" → Object1: "bear" (5x), Object2: "donut" (1x)

### 2. Always Use Regional Prompting
MUST use ConditioningSetArea or regional_prompt nodes even for simple 2-object scenes:
- Assign non-overlapping spatial regions (left/right split or foreground/background)
- Give each category its own conditioning area
- Use strength=1.0 for primary objects, 0.8-0.9 for background

### 3. Explicit Count Enforcement
For counted objects ("five bears", "six flowers"):
- Duplicate the object name: "bear, bear, bear, bear, bear" in the regional prompt
- Add count verification in negative prompt: "missing bears, fewer than five bears"
- Use grid layout hints: "arranged in a row", "spread across the scene"

### 4. Attribute Binding
Bind each attribute to its object using parentheses and emphasis:
- "(green backpack:1.3)" not "green, backpack"
- "(wooden:1.2) flower" not "wooden flower" (prevents wood texture bleeding)
- Place attribute+object in same regional conditioning block

### 5. Category Separation in Negative Prompt
Add cross-contamination prevention:
- "merged objects, hybrid creature, fused, blended categories"
- "backpack with animal features" (for backpack+animal scenes)
- "food-textured animal" (for food+animal scenes)

### 6. Spatial Clarity
Add explicit spatial language:
- "a green backpack on the left side, a pig on the right side"
- "five bears in the foreground, a donut in the background"
- Helps both regional prompting and global composition

## Example Workflow Pattern
```
CLIPTextEncode (global): "professional photo, clear focus, plain background"
CLIPTextEncode (region1): "(green backpack:1.3), leather texture, detailed straps"
CLIPTextEncode (region2): "(pig:1.3), pink skin, realistic animal"
ConditioningSetArea (region1): x=0, y=0, width=512, height=1024, strength=1.0
ConditioningSetArea (region2): x=512, y=0, width=512, height=1024, strength=1.0
ConditioningCombine: merge both regions
Negative: "merged objects, hybrid creature, fewer objects than specified, missing backpack, missing pig"
```

## Fallback
If regional prompting nodes unavailable, use:
- Heavy emphasis syntax: "(green backpack:1.4) AND (pig:1.4)"
- Explicit negative for the opposite: "no backpack-pig hybrid"
- Increase CFG to 8-9 for stronger prompt adherence

## Success Criteria
- Both/all categories present in output
- Correct count for enumerated objects
- Attributes correctly bound (green on backpack, not on pig)
- No visual merging or contamination between categories
